## Key Implementation Notes

### Resolving D-02 vs D-04 Conflict
User chose "memory cache" (D-02) but requires "show last successful data on error" (D-04). Memory cache doesn't persist across instances.

**Resolution per RESEARCH.md:** Use EdgeOne Pages KV storage instead of memory cache:
- Store `{ data, timestamp }` in KV
- Check `Date.now() - timestamp < 30min` on read
- On API error, return cached data regardless of age (with `stale: true` flag)

### Weather Condition Mapping
Map `weather[0].id` to Chinese per D-05:
- 800: 晴朗
- 801-802: 少云/多云
- 803-804: 阴
- 500-531: 雨/小雨/阵雨
- 600-622: 雪
- 200-232: 雷阵雨
- 701-781: 雾/霾

### Title Format (D-06)
`${city} ${temp}°C ${condition} - ${HH:MM}`
Example: `北京 22°C 晴朗 - 14:30`
